# Agent Engineering Canvas Local

El Agent Engineering Canvas Local es una herramienta portátil diseñada para transformar necesidades expresadas en lenguaje cotidiano en un Project Model trazable. Facilita la implementación de Loop Engineering mediante una interfaz gráfica intuitiva que integra los pilares de contexto, harness y loop, mejorando la experiencia del usuario y optimizando el uso de tokens durante el desarrollo de prompts.
